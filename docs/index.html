Radio Meteor Project

Detecting meteors by radio, from a cloudy city

An RTL-SDR listens on a locally vacant FM channel — 104.3 MHz — for the brief reflection of a distant, over-the-horizon transmitter off the ionized trail of a meteor. Each reflection is a “ping.” It works in daylight, and it works through the overcast that makes visual meteor watching hopeless in Seattle.

Station Seattle, WA  ·  Reference 104.3 MHz (KAWO Boise, ~650 km)  ·  Hardware RTL-SDR + Raspberry Pi 3B+  ·  code v1.5
